# Todo App v1

A clean Next.js starter for a todo list web app. This foundation keeps the architecture intentionally small: App Router, TypeScript, Tailwind CSS, and a static homepage shell that is ready for real todo features.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- ESLint

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Scripts

- `npm run dev` starts the local development server.
- `npm run build` creates the production build.
- `npm run start` runs the production server.
- `npm run lint` runs the lint checks.
